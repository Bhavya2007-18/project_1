Attention Guidance Sandbox

Self-Attention analyzes pixel-to-pixel correlations, while Cross-Attention links pixels to words. Select a prompt below, then hover over individual words to visualize the corresponding cross-attention weights mapping.

Attention Formula:
Attention(Q, K, V) = softmax( (Q Kᵀ) / √d_k ) V

Here, Queries ($Q$) represent image spatial features, Keys ($K$) correspond to token word embeddings, and Values ($V$) represent structural visual contents.
